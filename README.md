# BAELISH - Smart system for 'A Game Of Thrones' board game computer player
This project implements smart artificial player for board game "A Game of Thrones".

The player itself is not yet available.

## Set-up
Project dependencies are managed via [UV package manager](https://docs.astral.sh/uv/#projects).

To test the project, you can run
```
uv sync
uv run example.py
```
